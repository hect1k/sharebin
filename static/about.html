What is shareb.in?

shareb.in is a minimalist, open-source file, link, and text sharing utility. It's built on the principle of ephemeral data transfer - meaning your shared content is automatically and permanently deleted after a set period. We believe in simplicity, security, and giving you control over your temporary data.

The Philosophy: Suckless, but Usable

Inspired by the suckless philosophy, shareb.in is designed to be lightweight, fast, and unbloated. While the primary goal is simple utility over excessive features, we've ensured the platform remains highly usable for everyone, whether through the web interface or command line.

Focus on the Command Line (cURL)

For power users and developers, shareb.in excels as a CLI tool. You can effortlessly upload files or paste text directly from your terminal using common tools like cURL. Since anonymous usage offers very generous limits, it's perfect for quick scripting and piping data without needing to log in. View the full API documentation here: /redoc.

Key Features

  • Minimalist Sharing: File Share, URL Shortener, and Pastebin - nothing more, nothing less.
  • Automatic Expiry: All items are automatically purged from the server after their set time expires.
  • Open Source & Self-Hostable: You have the freedom to audit the source code, deploy your own instance, and truly control your temporary sharing needs.
  • Anonymous-Friendly: Excellent usage limits for anonymous sharing via the command line or web.
  • User Accounts: Registered users gain access to custom shortcodes, extended expiry options, and history tracking.
